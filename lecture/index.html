前端开发的职责就是让用户体验更友好。

前端开发工程师的主要工作就是把设计图按W3c标准做成html页面，并且用javascript脚本语言实现页面上的前端互动，以及前端优化。

{2 }Basic knowledge

Html

Html是一种超文本标记语言，不是编程语言。

一个最简单的HTML5文档需要的内容：<!DOCTYPE html>，保存为.html或.htm即可。

Css

层叠样式表，它可以改变html页面的样式，布局，使页面变得非常美观。

在CSS3中，它甚至可以完成很多炫酷的动画效果，而不需要冗长的js代码了。

Javascript

它是动态类型、弱类型、基于原型的语言。

它由 ECMAScript，DOM，BOM组成。

ECMAScript描述了语法和基本对象。

DOM描述处理网页内容的方法和接口。

BOM描述与浏览器进行交互的方法和接口。

Jquery

一个javascript库。它封装了很多js方法，使开发者能以更精简的代码做更多的事。

如果你不会js，学习它让你可以轻松完成任务。

IDE

Sublime Text、WebStorm、HBuilder、Dreamweaver

{3}Technologies

前端总有新的灵感和技术闪现出来，各种框架层出不穷。前端开发的入门门槛非常低，与服务器端语言先慢后快的学习曲线相比，前端开发的学习曲线是先快后慢，后面的学习曲线越来越陡峭，每前进一步都很难。

Learning Plan

如果你正在创建一个简单的 CRUD 程序，那么你就不需要使用 Relay；如果你正在学习 ES6，并不需要深入地了解 Async/Await或装饰器；如果你刚刚开始学习 React，并不需要使用热重载和服务端渲染；如果你刚刚接触 Webpack，你就不需要分离代码和合并多个资源；如果你刚刚学习 Redux，你不需要理解使用 Redux-Form 和 Redux-Sagas。

保持简洁，每次只做一件事！
